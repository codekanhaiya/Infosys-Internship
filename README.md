# 🚗 Capstone Project

# AI-Powered Car Lease & Loan Contract Review Assistant

The primary project developed during the internship is an intelligent contract review platform designed to simplify vehicle financing agreements.

Many consumers struggle to understand lengthy lease and loan documents.

This system acts as an AI-powered financial assistant capable of:

- Reviewing contracts
- Extracting important clauses
- Identifying risks
- Benchmarking market prices
- Assisting negotiations

---

# 🔥 Key Features

## 📄 Smart Contract Analysis

Users can upload:

- Lease Agreements
- Loan Contracts
- Finance Documents
- Scanned PDFs

The system automatically extracts:

- Interest Rate (APR)
- Loan Amount
- Monthly Payments
- Contract Duration
- Processing Charges
- Penalties
- Buyout Conditions
- Early Closure Terms

---

## 🔍 OCR Document Processing

The platform can process scanned documents using OCR technologies.

Capabilities include:

- Text Extraction
- PDF Parsing
- Clause Detection
- Structured Data Generation

---

## 🤖 LLM-Based Explanation Engine

Advanced language models help convert complex legal and financial terminology into user-friendly explanations.

The assistant can:

- Explain clauses
- Highlight risky conditions
- Answer finance-related questions
- Generate negotiation suggestions

---

## 💰 Market Value Benchmarking

The system evaluates vehicle pricing using available datasets and market information.

Users can compare:

- Market Value
- Lease Offers
- Loan Terms
- Dealer Pricing

---

## 🚘 Vehicle Information Analysis

Users can analyze:

- Vehicle Specifications
- Model Information
- Finance Details
- Ownership Considerations

---

## 💬 Negotiation Assistant

The AI assistant helps users:

- Identify unfair clauses
- Prepare negotiation points
- Draft responses
- Improve decision-making
